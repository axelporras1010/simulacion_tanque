Automatización de un tanque

Porras Axel C.I:29.545.523

Sánchez Francisco C.I: 29.545.027

Proceso a automatizar

Se requiere automatizar un tanque de agitado y calentado de un líquido. Al iniciar el proceso mediante un botón de inicio se vierte en el tanque el líquido hasta que el sensor del límite máximo permitido se activa, una vez llenado el tanque se procede a calentar el mismo hasta que alcance una temperatura superior a 100 grados además de agitarlo durante unos 10 segundos, al culminar dichos procesos, se abre una válvula de vaciado del tanque para realizar el proceso nuevamente cuando se requiera.

En base a estos requerimientos se realizo la siguiente transformación a lenguaje GRAFCET para luego poder hacer la simulación en la herramienta TwidoSuite
